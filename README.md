# Receta

### Sistema experto para asistencia en preparación de tortas de vainilla

## Instalar proyecto
```
npm install
```

### Compila y ejecuta servidor para hot-reloads en desarrollo
```
npm run serve
```

### Compila y minifica para producción
```
npm run build
```

### Lints y corrige archivos
```
npm run lint
```
